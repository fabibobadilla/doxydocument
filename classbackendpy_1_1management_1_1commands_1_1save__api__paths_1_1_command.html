Descripción detallada

Comando para guardar todas las rutas de la API en la base de datos.

Este comando obtiene todas las rutas definidas en el 
proyecto Django y las guarda en la tabla ApiPaths de la 
base de datos. Si se encuentra en la tabla, 
se eliminarán primero los registros existentes con 
backend_field=True para evitar duplicados.

@help: Mensaje que describe la funcionalidad del comando.
