Descripción detallada

Modelo que define una API.

Attributes:
    method (CharField): Método HTTP utilizado (GET, POST, etc.).
    serializer (CharField): Nombre del serializer utilizado.
    description (CharField): Descripción de la API.
    authenticatable (IntegerField): Indica si la API es autenticable.
    url_list (CharField): URL para listar los recursos de la API.
    url_path (CharField): Ruta de la URL para acceder a la API.
    fun_definition (CharField): Definición de la función de la API.
