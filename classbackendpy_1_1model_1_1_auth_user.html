Descripción detallada

Modelo que representa un usuario de autenticación.

Attributes:
    password (CharField): Contraseña del usuario.
    last_login (DateTimeField): Fecha y hora del último inicio de sesión.
    is_superuser (IntegerField): Indica si el usuario es un superusuario.
    username (CharField): Nombre de usuario único.
    first_name (CharField): Nombre del usuario.
    email (CharField): Correo electrónico del usuario.
    is_staff (IntegerField): Indica si el usuario tiene acceso al sitio administrativo.
    is_active (IntegerField): Indica si la cuenta del usuario está activa.
    date_joined (DateTimeField): Fecha y hora en que se unió el usuario.
