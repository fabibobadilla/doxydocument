Descripción detallada

Modelo que representa las restricciones de acceso a la API en el sistema Kaze.

Attributes:
    filter_table (CharField): Nombre de la tabla que se utilizará para filtrar los datos.
    sql_filter (CharField): Condición SQL para aplicar como filtro en las consultas.
    userid_restriction (IntegerField): Identificador del usuario al que se le aplica la restricción.

Meta:
    managed (bool): Indica si Django debe manejar la creación y eliminación de la tabla en la base de datos.
    db_table (str): Nombre de la tabla en la base de datos que se utiliza para este modelo.
