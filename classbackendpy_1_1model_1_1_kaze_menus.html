Descripción detallada

Modelo que representa los menús en el sistema Kaze.

Attributes:
    id (BigAutoField): Identificador único del menú.
    nombre (CharField): Nombre del menú.
    path (CharField): Ruta asociada al menú.
    remember_token (CharField): Token de recuerdo para el menú, si aplica (opcional).
    created_at (DateTimeField): Fecha y hora de creación del menú (opcional).
    updated_at (DateTimeField): Fecha y hora de la última actualización del menú (opcional).

Meta:
    managed (bool): Indica si Django debe manejar la creación y eliminación de la tabla en la base de datos.
    db_table (str): Nombre de la tabla en la base de datos que se utiliza para este modelo.
