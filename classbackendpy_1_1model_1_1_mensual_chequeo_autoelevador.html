Descripción detallada

Modelo que representa el chequeo mensual de un autoelevador.

Attributes:
    dni (IntegerField): Documento Nacional de Identidad del responsable del chequeo. (Opcional)
    fechahora (DateTimeField): Fecha y hora en que se realizó el chequeo. (Opcional)
    compruebe_tuercas_rueda (CharField): Indica si se comprobaron las tuercas de las ruedas. (Opcional)
    compruebe_tuercas_gral (CharField): Indica si se comprobaron las tuercas generales. (Opcional)
    realice_engrase_gral (CharField): Indica si se realizó el engrase general. (Opcional)
    observaciones (CharField): Observaciones adicionales sobre el chequeo. (Opcional)
    declaro_juradamente (CharField): Indica si se declaró juradamente la condición del autoelevador. (Opcional)

Meta:
    managed (bool): Indica si Django debe manejar la creación y eliminación de la tabla en la base de datos.
    db_table (str): Nombre de la tabla en la base de datos que se utiliza para este modelo.
