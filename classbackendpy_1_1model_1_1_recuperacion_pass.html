Descripción detallada

Modelo que representa la recuperación de contraseñas para los clientes autogestionados.

Attributes:
    id (IntegerField): Identificador único del registro de recuperación de contraseña.
    idclientes_autogestion (IntegerField): Identificador del cliente autogestionado asociado.
    token (CharField): Token utilizado para la recuperación de contraseña.
    fechahoraexp (DateTimeField): Fecha y hora de expiración del token.
    expirado (IntegerField): Indica si el token ha expirado (1 para sí, 0 para no).

Meta:
    managed (bool): Indica si Django debe manejar la creación y eliminación de la tabla en la base de datos.
    db_table (str): Nombre de la tabla en la base de datos que se utiliza para este modelo.
