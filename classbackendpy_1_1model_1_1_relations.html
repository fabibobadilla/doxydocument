Descripción detallada

Modelo que representa las relaciones entre diferentes tablas en la base de datos.

Attributes:
    id (IntegerField): Identificador único de la relación.
    description (CharField): Descripción de la relación.
    form_definition_from (CharField): Definición del formulario desde el que se establece la relación.
    table_from (CharField): Nombre de la tabla de origen de la relación.
    table_from_key (CharField): Clave de la tabla de origen utilizada para la relación.
    form_definition_to (CharField): Definición del formulario al que se establece la relación.
    table_to (CharField): Nombre de la tabla de destino de la relación.
    table_to_key (CharField): Clave de la tabla de destino utilizada para la relación.
    editable (IntegerField): Indica si la relación es editable (1 para sí, 0 para no).
    relation_type (CharField): Tipo de relación (por ejemplo, "1:1", "1:n", "n:m").
    table_api_schema (CharField): Esquema de la API asociado a la tabla.

Meta:
    managed (bool): Indica si Django debe manejar la creación y eliminación de la tabla en la base de datos.
    db_table (str): Nombre de la tabla en la base de datos que se utiliza para este modelo.
