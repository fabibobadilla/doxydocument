Descripción detallada

Modelo que representa la configuración de una tabla para la API.

Attributes:
    id (IntegerField): Identificador único del esquema de la tabla (clave primaria).
    table_key_name (CharField): Nombre clave de la tabla que se utilizará en la API.
    api_path (CharField): Ruta de la API asociada a la tabla.
    pagination (IntegerField): Indica el tamaño de la paginación para la tabla.
    editable (IntegerField): Indica si la tabla es editable (1 para editable, 0 para no editable).
    deletable (IntegerField): Indica si la tabla es eliminable (1 para eliminable, 0 para no eliminable).

Meta:
    managed (bool): Indica si Django debe manejar la creación y eliminación de la tabla en la base de datos.
    db_table (str): Nombre de la tabla en la base de datos que se utiliza para este modelo.
