Descripción detallada

Modelo que representa los parámetros de configuración de la aplicación.

Attributes:
    apiUrl (TextField): URL de la API utilizada por la aplicación.
    accessToken (TextField): Token de acceso para autenticación.
    refreshToken (TextField): Token de refresco para obtener nuevos tokens de acceso.
    clientId (TextField): Identificador del cliente para la autenticación.
    clientSecret (TextField): Secreto del cliente para la autenticación.
    wellKnown (TextField): URL de configuración para la autenticación.
    callbackUrl (TextField): URL de retorno utilizada en la autenticación.
    scopes (TextField): Alcances de acceso solicitados.
    orgConnectionCompletedUrl (TextField): URL a la que se redirige una vez completada la conexión de la organización.
    state (TextField): Estado de la solicitud, utilizado para la validación de respuestas.
    apiResponse (TextField): Respuesta de la API tras la autenticación o solicitud.
    accessTokenDetails (TextField): Detalles adicionales sobre el token de acceso.
    exp (TextField): Tiempo de expiración del token de acceso.
    idToken (TextField): Token de identificación del usuario.

Meta:
    managed (bool): Indica si Django debe manejar la creación y eliminación de la tabla en la base de datos.
    db_table (str): Nombre de la tabla en la base de datos que se utiliza para este modelo.
