Descripción detallada

Serializador base que permite campos opcionales.

Este serializador permite que todos los campos sean opcionales y puedan
aceptar valores nulos o vacíos. Se utiliza como clase base para otros
serializadores.

Attributes:
    fields (dict): Un diccionario de campos del modelo que se están
    serializando.
