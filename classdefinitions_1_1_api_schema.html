Descripción detallada

Class representing the API response schema.

Attributes:
    result (bool): Indicates if the operation was successful.
    message (str): Message associated with the response.
    status (int): HTTP status code.
    datetime (datetime): Timestamp of the response.
    content_type (str): Type of content ("object" or "list").
    content (dict): JSON content of the response.

Methods:
    error_response(reason, status): Generates an error response.
    succesful_response(serializer_data, mensaje): Generates a successful response.
