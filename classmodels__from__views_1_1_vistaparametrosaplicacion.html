Descripción detallada

Modelo para representar los parámetros de la aplicación.

Atributos:
    apiUrl (str): URL de la API.
    accessToken (str): Token de acceso.
    refreshToken (str): Token de actualización.
    clientId (str): ID del cliente.
    clientSecret (str): Secreto del cliente.
    wellKnown (str): URL well-known para la configuración de OpenID Connect.
    callbackUrl (str): URL de callback para la autenticación.
    scopes (str): Alcances solicitados para la autenticación.
    orgConnectionCompletedUrl (str): URL que se carga después de que la conexión con la organización se completa.
    state (str): Estado de la autenticación.
    apiResponse (str): Respuesta de la API.
    accessTokenDetails (str): Detalles del token de acceso.
    exp (str): Tiempo de expiración del token.
    idToken (str): Token de identidad.
