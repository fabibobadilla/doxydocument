Descripción detallada

Modelo para representar la vista de los usuarios CSC.

Atributos:
    id (str): Identificador único del usuario.
    email (str): Correo electrónico del usuario.
    pass (str): Contraseña del usuario.
    nombre (str): Nombre del usuario.
    usersysid (str): ID del sistema del usuario.
    activo (str): Estado de actividad del usuario.
    last_index (str): Último índice.
    admin (str): Indica si el usuario es administrador.
    id_responsable_servicios (str): ID del responsable de servicios.
    supervisor (str): Indica si el usuario es supervisor.
    general (str): Campo general.
    sucursal (str): Sucursal a la que pertenece el usuario.
    atendiendo (str): Indica si el usuario está atendiendo.
    fechahorainiciorevision (str): Fecha y hora de inicio de la revisión.
    ultimafechahorarevision (str): Última fecha y hora de revisión.
    minutosextension (str): Minutos de extensión.
    whatsapp (str): Número de WhatsApp del usuario.
