Descripción detallada

CREATE. 

Vista para crear un nuevo usuario.

Esta vista permite la creación de un nuevo registro en el modelo 
Usuarios, generando un nuevo salt y almacenando la contraseña 
hasheada asociada.

@queryset Usuarios.objects.all(): 
    Conjunto de consultas de todos los usuarios.

@serializer_class UsuariosSerializer: 
    Serializador utilizado para validar y deserializar los datos de entrada.

@perform_create: 
    Método que se llama cuando se crea un nuevo usuario. 
    Genera un nuevo salt, hashea la contraseña y guarda el usuario 
    junto con su salt.

@memberof UsuariosCreateAPIView
