acciones_email_alerts.get_acciones_email_alerts ( request,
pk = None ) 
Obtiene las acciones de alertas de correo electrónico.

Esta vista permite obtener una lista de todas las acciones de alertas de correo electrónico
o, si se proporciona un parámetro pk, obtener una acción específica.

Args:
    request: Objeto de solicitud HTTP.
    pk (int, opcional): Identificador de la acción específica a obtener.

Returns:
    Response: Respuesta JSON con los datos de las acciones de alertas de correo electrónico
              o un mensaje de error si no se encuentra el recurso.
