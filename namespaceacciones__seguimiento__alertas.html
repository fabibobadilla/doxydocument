acciones_seguimiento_alertas.get_acciones_seguimiento_alertas ( request,
pk = None ) 
Obtiene las acciones de seguimiento de alertas.

Esta vista permite obtener una lista de todas las acciones de seguimiento de alertas
o, si se proporciona un parámetro pk, obtener una acción específica.

Args:
    request: Objeto de solicitud HTTP.
    pk (int, opcional): Identificador de la acción específica a obtener.

Returns:
    Response: Respuesta JSON con los datos de las acciones de seguimiento de alertas
              o un mensaje de error si no se encuentra el recurso.
