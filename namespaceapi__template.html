api_template.TABLE_NAME ( request,
pk = None ) 
Obtiene una lista de instancias de <MODEL_NAME>.

Si se proporciona el parámetro `pk`, devuelve la instancia específica de <MODEL_NAME> correspondiente a esa clave primaria.

Parameters:
    request (HttpRequest): La solicitud HTTP que contiene los parámetros.
    pk (int, optional): La clave primaria de la instancia específica que se desea obtener. Por defecto es None.

Returns:
    Response: Un objeto Response que contiene:
        - 200: La lista de instancias de <MODEL_NAME> o la instancia específica si se proporciona `pk`.
        - 400: Un error de solicitud incorrecta si se produce un problema al obtener los datos.
        - 404: Un error de no encontrado si no se encuentra la instancia específica con el `pk` proporcionado.
