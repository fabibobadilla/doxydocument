◆ get_auth_tokens()

auth_tokens.get_auth_tokens ( request,
pk = None ) 
Vista que obtiene los datos de AuthTokens.

Dependiendo de si se proporciona o no un parámetro pk, esta vista devuelve:
- Una lista completa de todos los AuthTokens.
- Un único AuthTokens si se proporciona el parámetro pk.

Args:
    request: El objeto de la solicitud HTTP.
    pk (int, opcional): El identificador único del AuthTokens específico.

Returns:
    Response: Un objeto Response con los datos serializados de AuthTokens o una lista de ellos.
