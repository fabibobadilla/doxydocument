auth_user.get_auth_user ( request,
pk = None ) 
Vista que obtiene los datos de AuthUser.

Dependiendo de si se proporciona o no un parámetro pk, esta vista devuelve:
- Una lista completa de todos los AuthUser.
- Un único AuthUser si se proporciona el parámetro pk.

Args:
    request: El objeto de la solicitud HTTP.
    pk (int, opcional): El identificador único del AuthUser específico.

Returns:
    Response: Un objeto Response con los datos serializados de AuthUser o una lista de ellos.
