◆ get_campaigns()

campaigns.get_campaigns ( request,
pk = None ) 
Vista que obtiene los datos de Campaigns.

Dependiendo de si se proporciona o no un parámetro pk, esta vista devuelve:
- Una lista completa de todos los Campaigns.
- Un único Campaigns si se proporciona el parámetro pk.

Args:
    request: El objeto de la solicitud HTTP.
    pk (int, opcional): El identificador único del Campaigns específico.

Returns:
    Response: Un objeto Response con los datos serializados de Campaigns o una lista de ellos.
