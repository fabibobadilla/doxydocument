◆ get()

common.get ( request) 
Obtiene una lista de todos los nombres de las tablas en la base de datos.

Este endpoint permite recuperar el nombre de todas las tablas existentes en la base de datos.

Parámetros:
    request (Request): La solicitud HTTP que contiene los datos.

Retorna:
    Response: Un objeto Response que contiene un diccionario con los nombres de las tablas
              en el campo "results".

◆ get_columns()

common.get_columns ( request) 
Obtiene una lista de las columnas de una tabla específica.

Este endpoint permite recuperar los nombres de las columnas de la tabla indicada en la
solicitud.

Parámetros:
    request (Request): La solicitud HTTP que contiene el nombre de la tabla en el cuerpo.

Retorna:
    Response: Un objeto Response que contiene un diccionario con los nombres de las columnas
              en el campo "results". Si no se proporciona el nombre de la tabla, se devuelve
              un array vacío.
