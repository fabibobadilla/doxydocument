corsvap_derivaciones.get_corsvap_derivaciones ( request,
pk = None ) 
Vista que obtiene los datos de CorsvapDerivaciones.

Dependiendo de si se proporciona o no un parámetro pk, esta vista devuelve:
- Una lista completa de todos los CorsvapDerivaciones.
- Un único CorsvapDerivaciones si se proporciona el parámetro pk.

Args:
    request: El objeto de la solicitud HTTP.
    pk (int, opcional): El identificador único de la CorsvapDerivaciones específica.

Returns:
    Response: Un objeto Response con los datos serializados de CorsvapDerivaciones o una lista de ellos.
