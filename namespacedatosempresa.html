datosempresa.get_datosempresa ( request,
pk = None ) 
Vista que obtiene los datos de Datosempresa.

Dependiendo de si se proporciona o no un parámetro pk, esta vista devuelve:
- Una lista completa de todos los Datosempresa.
- Un único Datosempresa si se proporciona el parámetro pk.

Args:
    request: El objeto de la solicitud HTTP.
    pk (int, opcional): El identificador único de la Datosempresa específica.

Returns:
    Response: Un objeto Response con los datos serializados de Datosempresa o una lista de ellos.
