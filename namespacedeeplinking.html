◆ get_deeplinking()

deeplinking.get_deeplinking ( request,
pk = None ) 
Vista que obtiene los datos de Deeplinking.

Dependiendo de si se proporciona o no un parámetro pk, esta vista devuelve:
- Una lista completa de todos los Deeplinking.
- Un único Deeplinking si se proporciona el parámetro pk.

Args:
    request: El objeto de la solicitud HTTP.
    pk (int, opcional): El identificador único del Deeplinking específico.

Returns:
    Response: Un objeto Response con los datos serializados de Deeplinking o una lista de ellos.
