etags_jd.get_etags_jd ( request,
pk = None ) 
Vista que obtiene los datos de EtagsJd.

Dependiendo de si se proporciona o no un parámetro pk, esta vista devuelve:
- Una lista completa de todos los EtagsJd.
- Un único EtagsJd si se proporciona el parámetro pk.

Args:
    request: El objeto de la solicitud HTTP.
    pk (int, opcional): El identificador único del EtagsJd específico.

Returns:
    Response: Un objeto Response con los datos serializados de EtagsJd o una lista de ellos.
