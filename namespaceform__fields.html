◆ generate_form_fields_table()

form_fields.generate_form_fields_table ( request) 
Vista API para generar campos de formulario basados en una tabla SQL.

Este endpoint recibe el nombre de una tabla SQL y devuelve una lista de
campos de formulario que se pueden utilizar para crear o editar formularios
basados en esa tabla.

Args:
    request (HttpRequest): La solicitud HTTP que contiene los parámetros de consulta, incluyendo el nombre de la tabla SQL.

Returns:
    Response: Un objeto JSON con la lista de campos de formulario generados.

◆ get_form_fields()

form_fields.get_form_fields ( request,
pk = None ) 
Vista API para obtener `FormFields`.

Esta vista maneja dos escenarios:
1. Si se proporciona un `pk` en la solicitud, se devuelve un campo de formulario específico.
2. Si no se proporciona `pk`, se devuelve una lista con todos los campos de formulario disponibles.

Args:
    request (HttpRequest): La solicitud HTTP que contiene los parámetros de consulta.
    pk (int, opcional): ID del campo de formulario específico a recuperar. 

Returns:
    Response: Un objeto JSON con los datos del campo de formulario específico o una lista de ellos.

Raises:
    Http404: Si el `FormFields` con el `pk` proporcionado no existe.
