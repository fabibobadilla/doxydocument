◆ get_forms()

forms.get_forms ( request,
pk = None ) 
Obtiene una lista de formularios (`Forms`) o un formulario específico según el parámetro `pk`.

Args:
    request (HttpRequest): La solicitud HTTP.
    pk (int, optional): ID del formulario que se desea obtener. Si no se proporciona, se devolverán todos los formularios.

Returns:
    Response: Un objeto Response con los datos del formulario o formularios obtenidos.

Raises:
    Http404: Si el `Forms` con el `pk` proporcionado no existe.
