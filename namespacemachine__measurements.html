machine_measurements.get_and_save_machine_measurements ( request) 
Vista que obtiene y guarda las mediciones de la máquina en la base de datos.

Esta vista realiza una solicitud GET a una URL específica, procesa los datos obtenidos y guarda las mediciones
en la base de datos, eliminando las mediciones previas del mismo pin y período.

Args:
    request: El objeto de la solicitud HTTP que contiene los datos de entrada.

Returns:
    Response: Un objeto Response que contiene el resultado del procesamiento de las mediciones o un error.

◆ get_machine_measurements()

machine_measurements.get_machine_measurements ( request,
pk = None ) 
Vista que obtiene los datos de MachineMeasurements.

Dependiendo de si se proporciona o no un parámetro pk, esta vista devuelve:
- Una lista completa de todos los MachineMeasurements.
- Un único MachineMeasurements si se proporciona el parámetro pk.

Args:
    request: El objeto de la solicitud HTTP.
    pk (int, opcional): El identificador único del MachineMeasurements específico.

Returns:
    Response: Un objeto Response con los datos serializados de MachineMeasurements o una lista de ellos.
