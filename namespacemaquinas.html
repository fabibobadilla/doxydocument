maquinas.get_maquinas ( request,
pk = None ) 
Vista que obtiene los datos de Maquinas.

Dependiendo de si se proporciona o no un parámetro pk, esta vista devuelve:
- Una lista completa de todos los Maquinas.
- Un único Maquinas si se proporciona el parámetro pk.

Args:
    request: El objeto de la solicitud HTTP.
    pk (int, opcional): El identificador único del Maquinas específico.

Returns:
    Response: Un objeto Response con los datos serializados de Maquinas o una lista de ellos.

◆ obtener_datos_maquinarias()

maquinas.obtener_datos_maquinarias ( request) 
Obtiene los datos de maquinarias a partir de una solicitud HTTP.

Esta función procesa la solicitud, valida el token de autenticación y obtiene los datos 
proporcionados en el cuerpo de la solicitud. Si el token no es válido, devuelve una respuesta de error.

Args:
    request (HttpRequest): 
        El objeto de solicitud que contiene los datos de la solicitud HTTP.

Returns:
    Response: 
        Una respuesta de error si el token es inválido, de lo contrario, 
        puede devolver los datos de maquinarias obtenidos (dependiendo de la implementación).

Raises:
    ValueError: 
        Si el cuerpo de la solicitud no se puede deserializar como JSON.

Example:
    >>> request = HttpRequest()
    >>> obtener_datos_maquinarias(request)
