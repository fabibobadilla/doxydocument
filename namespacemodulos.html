◆ get_modulos()

modulos.get_modulos ( request,
pk = None ) 
Vista que obtiene los datos de Modulos.

Dependiendo de si se proporciona o no un parámetro pk, esta vista devuelve:
- Una lista completa de todos los Modulos.
- Un único Modulo si se proporciona el parámetro pk.

Args:
    request: El objeto de la solicitud HTTP.
    pk (int, opcional): El identificador único del Modulo específico.

Returns:
    Response: Un objeto Response con los datos serializados de Modulos o una lista de ellos.
