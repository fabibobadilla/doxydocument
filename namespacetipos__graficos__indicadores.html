tipos_graficos_indicadores.get_tipos_graficos_indicadores ( request,
pk = None ) 
Obtiene una lista de tipos de gráficos indicadores o un tipo específico por ID.

Si se proporciona el parámetro 'pk', se devuelve el objeto correspondiente a ese ID. 
Si no se proporciona 'pk', se devuelve una lista de todos los tipos de gráficos indicadores.

Args:
    request (HttpRequest): La solicitud HTTP recibida.
    pk (int, optional): El ID del tipo de gráfico indicador a obtener. Por defecto es None.

Returns:
    Response: Un objeto de respuesta que contiene los datos de los tipos de gráficos indicadores.
              - Si 'pk' es proporcionado, devuelve el objeto específico.
              - Si 'pk' no es proporcionado, devuelve una lista de objetos.
